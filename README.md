
# 🎇 弹力球打砖块 

[体验地址](https://brick.qizhen.xyz/)

一款基于 **Phaser 3** 构建的极具现代视觉冲击力与深度物理优化的打砖块游戏。本作复刻和改进了经典的打砖块游戏。




## 🌟 核心特色

* **极致视觉美学**：
* **自适应辉光 (Bloom)**：基于 Phaser FX Pipeline 实现的实时光效，渲染出纯正的霓虹质感。
* **动态拖尾系统**：小球尾迹颜色与长度随速度动态变化，提供极佳的运动反馈。
* **零资源依赖**：所有物体（球体、挡板、背景）均由代码动态绘制纹理，确保极小的首屏加载体积。


* **多样化道具 (8+ Power-ups)**：
* 🛡️ **挡板增强**：伸缩变形，适应不同关卡节奏。
* 🔥 **战斗辅助**：火球穿透、多球分裂（Multi-ball）挑战手速极限。
* ⏳ **时空控制**：速度增益/减益，精准把控战场。


* **全平台优化**：针对移动端实现了 **Relative Displacement (相对偏移)** 操作，解决遮挡屏幕痛点。

---

## 🛠️ 技术深度

本项目不仅是一个游戏原型，更是对 Web 游戏物理瓶颈的一次探索：

### 1. 物理稳定性 (CCD & Sub-stepping)

针对高速碰撞下的“穿模”问题，本项目实现了高频率物理步进检查，确保在小球速度极快时仍能精准检测薄片砖块的边缘。

### 2. 逻辑与帧率解耦 (Fixed Timestep)

为了适配现代 120Hz/144Hz 高刷屏，挡板移动和补间动画采用 delta 时间补偿，确保不同设备上的游戏平衡性完全一致。

### 3. 程序化合成音频

摒弃庞大的 MP3 文件，利用 **Web Audio API** 实时合成振荡器信号，实现撞击声的实时变调与动态混响，体积近乎为零。

## 🎮 操作说明

| 动作 | PC 端 (Keyboard/Mouse) | 移动端 (Touch) |
| --- | --- | --- |
| **移动挡板** | `A`/`D` 或 鼠标水平移动 | 屏幕任意位置滑动 (盲操支持) |
| **发射球/开始** | `空格键` 或 鼠标左键 | 点击屏幕任意位置 |
| **暂停/菜单** | `ESC` 或 `P` | 点击右上角暂停图标 |

## 📦 快速开始

```bash
# 克隆仓库
git clone https://github.com/ruanqizhen/brick.git

# 安装依赖
npm install

# 启动开发服务器 (Vite)
npm run dev

# 构建生产版本
npm run build

```
