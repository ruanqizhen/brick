
# 🎇 弹力球打砖块 

[体验地址](https://brick.qizhen.xyz/)

一款基于 **Phaser 3** 构建的极具现代视觉冲击力与深度物理优化的打砖块游戏。本作复刻和改进了经典的打砖块游戏。


## 🎮 游戏指南

### 1. 基础玩法
* **目标**：控制挡板反弹小球，消除屏幕上所有的**普通砖块**即可过关。
* **难度模式**：
    * **简单模式**：基础球速，道具为传感器（小球穿过），适合新手。
    * **困难模式**：球速提升 50%，道具变为**物理实体**（小球会与道具碰撞），极具挑战性。
* **生命值**：初始拥有 3 颗心。如果小球掉出屏幕底端，将扣除 1 点生命值。生命归零则游戏结束。
* **发射**：关卡开始或丢球后，点击屏幕或按`空格`发射小球。

### 2. 砖块图鉴 (Bricks)
游戏包含多种性能各异的砖块，合理利用它们的特性是通关关键：

| 类型 | 外观 | 特性说明 |
| :--- | :--- | :--- |
| **基础砖块** | 🟦 青色 | 能量核心，击中 **1次** 即可消除。 |
| **强化砖块** | 🟨 黄色 | 拥有双层装甲，需要击中 **2次** 消除。 |
| **重装砖块** | 🟥 红色 | 最坚固的核心，需要击中 **3次** 消除。 |
| **爆炸砖块** | 💥 橙红 | 击碎后产生连锁反应，炸毁周围 **3x3** 范围内的所有砖块。 |
| **移动砖块** | 🔮 紫色 | 在空中水平来回游走，干扰你的瞄准轨迹。 |
| **隐形砖块** | 👻 半透 | 首次被击中前处于隐匿状态，击中后显现真实形态。 |
| **金属砖块** | 🔗 银灰 | 坚不可摧的阻挡物，无法消除，仅用于借力反弹。 |

### 3. 强力道具 (Power-ups)
击碎砖块有概率掉落功能多样的道具，接住它们可获得限时增益：

* ↔️ **挡板变长/变短**：改变防御面积，适应不同密度的砖块区。
* 🌕 **小球变大/变小**：改变碰撞体积。
* 🚀 **加速/减速**：控制游戏节奏。减速有助于精准接球，加速可快速清场。
* 🔥 **烈火球 (Fireball)**：小球处于燃烧状态，能**一击必杀**除了金属块以外的所有砖块。
* 💠 **分裂球 (Multi-ball)**：瞬间将场上所有球倍增，挑战多线火力的操控极限。
* 💖 **额外部命 (Extra Life)**：直接增加 1 点生命上限。

## 💡 进阶策略 (Strategy Tips)

1. **角度大师**：根据小球击中挡板的位置（左侧或右侧），反弹的角度会发生变化。利用这一点来避开障碍物或寻找缝隙。
2. **爆炸连锁**：优先攻击爆炸砖块，它可以替你清理大量高血量砖块，节省时间。
3. **分屏操作**：在移动端，挡板支持“盲操”，你可以按住屏幕下方任意位置左右滑动。
4. **火球突击**：烈火球是通关利器，它不会发生法线反弹而是直接穿透并摧毁砖块。

## 🛠️ 技术深度

本项目不仅是一个游戏原型，更是对 Web 游戏物理瓶颈的一次探索：

### 1. 物理引擎 (Matter.js 深度集成)
采用了 **Matter.js** 高级物理引擎，实现了高精度的碰撞检测与复杂的实体互动（如困难模式下的道具碰撞）。

### 2. 逻辑与帧率解耦 (Adaptive 120Hz+)
针对现代 120Hz/144Hz 高刷屏进行了针对性优化，物理歩进与渲染帧率解耦，确保不同频率下的运动手感完全一致。

### 3. 全局现代化布局
全新的三面板对称式 HUD 设计，采用磨砂玻璃（Frosted Glass）质感，适配各种屏幕尺寸，提供顶级的视觉一致性。
