
# 🎇 弹力球打砖块 

[体验地址](https://brick.qizhen.xyz/)

一款基于 **Phaser 3** 构建的极具现代视觉冲击力与深度物理优化的打砖块游戏。本作复刻和改进了经典的打砖块游戏。




## 🌟 核心特色

* **极致视觉美学**：
    * **自适应辉光 (Bloom)**：基于 Phaser FX Pipeline 实现的实时光效，渲染出纯正的霓虹质感。
    * **动态拖尾系统**：小球尾迹颜色与长度随速度动态变化，提供极佳的运动反馈。
    * **程序化矢量渲染**：所有道具图标、特殊砖块效果均由代码直接绘制矢量图形，彻底告别 Emoji 和图片，确保 100% 锐利度及极速加载。

* **多样化机制**：
    * **8+ 动态道具**：包括挡板伸缩、火球穿透、多球分裂（Multi-ball）、时空减速等叠加逻辑。
    * **智力特殊砖块**：
        * 💥 **爆炸砖块**：引发 3x3 范围连锁反应。
        * 👻 **隐形砖块**：动态干扰，首次击中显影。
        * 🚀 **移动砖块**：正弦曲线游走，由于 CCD 物理引擎支持，确保高速碰撞无误。

* **全平台优化**：针对移动端实现了 **Relative Displacement (相对偏移)** 操作，解决遮挡屏幕痛点。同时内置 **原生帮助场景 (Help Scene)**，图文并茂展示所有机制。

---

## 🛠️ 技术深度

本项目不仅是一个游戏原型，更是对 Web 游戏物理瓶颈的一次探索：

### 1. 物理稳定性 (CCD & Sub-stepping)
针对高速碰撞下的“穿模”问题，本项目实现了高频率物理步进检查与射线检测逻辑，确保在小球速度极快或砖块极薄时仍能精准检测碰撞。

### 2. 逻辑与帧率解耦 (Adaptive 120Hz+)
为了适配现代 120Hz/144Hz 高刷屏，物理引擎与渲染逻辑完全解耦，确保不同设备上的物理表现一致。

### 3. 极简依赖
完全移除了 `matter-js` 等大型物理引擎，纯手写 Arcade 物理深度扩展，打包体积压缩到极致。

## 📦 快速开始

```bash
# 克隆仓库
git clone https://github.com/ruanqizhen/brick.git

# 安装依赖
npm install

# 启动开发服务器 (Vite)
npm run dev

# 构建生产版本
npm run build
```
